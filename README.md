# AI Resume Summary Generator

This is a simple AI-based resume summary generator made using
React, Node.js and Express.

The app takes basic user details and generates a short
professional resume summary using a free AI API.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- AI API: OpenRouter (Free Model)

## Features
- Enter name, skills and experience
- Generate resume summary using AI
- Simple UI
- No database used

## How to Run Project

### Backend
1. Go to backend folder
2. Run `npm install`
3. Create `.env` file and add API key
4. Run `npm start`

### Frontend
1. Go to frontend folder
2. Run `npm install`
3. Run `npm start`

## Note
This project is made for learning purposes and beginner practice.
